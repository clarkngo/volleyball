# volleyball

**Homepage:** open **`index.html`** for links to all prototypes (works on **GitHub Pages** from repo root).

3D direction: **Three.js** prototypes and plan live under:

- **`docs/THREE_JS_PLAN.md`** — roadmap and stack notes  
- **`prototypes/03-standard-volleyball-court.html`** — **FIVB-scale** 18×9 m court, **neon night** palette (black void + cyan / magenta / lime / violet lines)  
- **`prototypes/01-court-scene.html`** — stylized court, same neon-night vibe  
- **`prototypes/02-ball-arc.html`** — serve arc + replay, neon trail + net glow  
- **`prototypes/04-player-positions.html`** — twelve stylized players, **USAV court zones Z1–Z6** (RB/RF/MF/LF/LB/MB), **CSS2D** labels, 3 formations  
- **`prototypes/05-outside-hitter-defense.html`** — **perimeter vs rotational** D vs OH, animated shifts, distinct roles, **responsibility rings**, **attack cone + wedge + arc**  

Open **`index.html`** or any prototype in a browser (needs network for the Three.js CDN).
