# Pre-write (before you build)

Short checks **before** a new prototype or doc lands:

- **`docs/IDEAS.md`** — Skim the backlog so you do not duplicate a planned or already-shipped idea (see **`index.html`** for what exists).
- **`docs/THREE_JS_PLAN.md`** — Direction for Three.js prototypes (stack, phases).
- **`README.md`** — Paths and naming should stay consistent when you add a new `prototypes/NN-*.html`.

After shipping, use **`docs/POST_WRITE.md`** and append **`CHANGELOG.md`**.
